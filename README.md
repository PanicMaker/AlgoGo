# AlgoGo

这个仓库包含了各种算法和数据结构的 Go 语言实现，并按照类别进行了组织。

## 类别

- Array (数组)
- BackTrack (回溯)
- BinarySearch (二分查找)
- BitManipulation (位运算)
- DataStructure (数据结构)
- DP (动态规划)
- Graph (图)
- Greddy (贪心)
- HashMap (哈希表)
- LinkedList (链表)
- Math (数学)
- NowCoder (牛客网)
- Queue (队列)
- SlidingWindow (滑动窗口)
- Sort (排序)
- Stack (栈)
- String (字符串)
- Tree (树)
